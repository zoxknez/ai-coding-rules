# Ruby Structure

> Gem layout with minimal folders.

## Canonical Layout

```text
project/
├─ my_gem.gemspec
├─ Gemfile
├─ README.md
├─ lib/
│  ├─ my_gem.rb
│  └─ my_gem/
│     └─ version.rb
├─ bin/
└─ spec/
```
