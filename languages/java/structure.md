# Java Structure

> Maven or Gradle standard layout with layers.

## Canonical Layout

```text
project/
├─ pom.xml
├─ README.md
├─ src/
│  ├─ main/
│  │  ├─ java/com/example/app/
│  │  │  ├─ Application.java
│  │  │  ├─ config/
│  │  │  ├─ domain/
│  │  │  ├─ application/
│  │  │  ├─ infrastructure/
│  │  │  └─ api/
│  │  └─ resources/
│  └─ test/
│     ├─ java/
│     └─ resources/
```
